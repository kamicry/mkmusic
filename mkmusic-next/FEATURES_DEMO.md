# 新功能演示指南

## 🎉 本次更新的新功能

### 1. 静默搜索 🔍

**之前:** 搜索时会弹出"搜索中"提示框  
**现在:** 搜索过程静默进行，无干扰

**体验方式:**
1. 点击搜索按钮
2. 输入歌曲名称
3. 点击"搜索"
4. 注意：不会再有"搜索中"的弹窗
5. 结果会直接显示在列表中

---

### 2. 歌词高亮显示 ✨

**效果:** 当前播放的歌词行会特别突出显示

**视觉效果:**
- 🟢 绿色文字（#31c27c）
- 📏 字体稍大（18px）
- 💪 加粗显示（font-weight: 500）
- ✨ 发光效果（text-shadow）
- 🔍 轻微放大（scale 1.05）

**体验方式:**
1. 播放任意有歌词的歌曲
2. 观察歌词显示区域
3. 当前播放的那一行会：
   - 变成绿色
   - 字体变大
   - 有发光效果
   - 轻微放大
4. 歌词会自动滚动，保持当前行在中央

**推荐测试歌曲:**
- 搜索"周杰伦 七里香"（网易云）
- 搜索"薛之谦 演员"（网易云）
- 搜索"邓紫棋 光年之外"（网易云）

---

### 3. 自动封面加载 🖼️

**功能:** 播放歌曲时自动获取并显示专辑封面

**显示位置:**
1. **播放器封面区** - 左上角的专辑图
2. **背景模糊层** - 整个页面的背景
3. **歌曲详情** - 详情面板中的封面预览

**体验方式:**
1. 播放任意歌曲
2. 观察封面加载：
   - 播放器左侧出现专辑图
   - 背景变成模糊的封面图
3. 点击详情按钮可以看到完整封面

**封面来源:**
- 从GD Studio API自动获取
- 使用高质量图片（500px）
- 支持网易云、酷我、JOOX

---

### 4. 歌曲详情面板 📋

**核心功能:** 一键查看歌曲所有详细信息和资源链接

#### 如何打开详情

**方式一：点击ℹ️按钮**
1. 在音乐列表中，每首歌曲右侧有一个 ℹ️ 图标
2. 点击图标（注意：不会触发播放）
3. 弹出详情面板

**方式二：快捷键**（未来可能添加）
- 选中歌曲后按 'i' 键

#### 详情面板内容

##### 📌 基本信息
```
🎵 歌曲名称: 七里香
🎤 艺术家: 周杰伦
💿 专辑: 七里香
🎵 音乐源: 网易云
🆔 歌曲ID: 185668
```

##### 🎼 音频信息
```
🎚️ 请求码率: 320kbps
📊 实际码率: 320kbps
💾 文件大小: 8.45 MB
```

##### 🔗 资源链接

**1. 音乐链接**
```
[输入框显示完整URL] [复制]
```
- 可以直接播放的音乐URL
- 点击"复制"按钮一键复制
- 点击输入框可以全选

**2. 封面链接**
```
[输入框显示完整URL] [复制]
```
- 高清专辑封面图片URL
- 可以在浏览器中直接打开
- 用于下载或分享封面

**3. 歌词API链接**
```
[输入框显示完整URL] [复制]
```
- GD Studio API的歌词接口
- 返回LRC格式歌词
- 可以用于开发或调试

##### 📷 封面预览

如果歌曲有封面，会显示一个大图预览：
```
┌─────────────────────┐
│                     │
│   [专辑封面图片]    │
│                     │
└─────────────────────┘
```

##### ℹ️ 使用说明

面板底部会显示三条重要提示：
1. ⏱️ 音乐链接有时效性，过期后需要重新获取
2. 🎚️ 实际码率可能与请求码率不同，取决于音源可用性
3. 🌐 所有链接均来自 GD Studio API

#### 交互功能

**复制链接:**
1. 点击任意"复制"按钮
2. 链接自动复制到剪贴板
3. 弹出提示："音乐链接已复制到剪贴板"

**手动复制:**
1. 点击输入框
2. 文本会自动全选
3. Ctrl+C (Windows) 或 Cmd+C (Mac) 复制

**关闭面板:**
- 方式1：点击右上角 ✕ 按钮
- 方式2：点击面板外的灰色背景
- 方式3：按 ESC 键（可能需要未来实现）

#### 样式特点

**标题栏:**
- 紫色渐变背景（#667eea → #764ba2）
- 白色文字
- 可以拖动（理论上，需要实现）

**内容区:**
- 清晰的分组标题（带emoji）
- 标签对齐（左对齐）
- 值对齐（右对齐）
- 舒适的间距

**按钮:**
- 绿色主题（#31c27c）
- 悬停时颜色加深
- 悬停时轻微上移
- 点击时有反馈

**响应式:**
- 桌面：600px宽度
- 移动：95%宽度
- 自动适配布局

## 🎯 完整使用流程

### 场景1：发现好歌并获取链接

1. **搜索歌曲**
   ```
   打开搜索 → 输入"周杰伦 告白气球" → 点击搜索
   ```

2. **播放试听**
   ```
   点击搜索结果 → 歌曲开始播放
   ```

3. **查看歌词**
   ```
   观察右侧歌词区 → 当前行高亮显示 → 跟着歌词唱
   ```

4. **获取资源**
   ```
   点击歌曲右侧ℹ️ → 打开详情面板
   → 复制音乐链接 → 复制封面链接
   → 用于分享或下载
   ```

### 场景2：批量收集歌曲信息

1. **搜索专辑**
   ```
   搜索"周杰伦 叶惠美"
   ```

2. **逐个查看**
   ```
   对每首感兴趣的歌：
   - 点击ℹ️按钮
   - 复制所需链接
   - 关闭详情
   - 继续下一首
   ```

3. **整理收藏**
   ```
   将复制的链接整理到文本文件
   或导入到其他音乐管理工具
   ```

### 场景3：分享音乐

1. **找到想分享的歌曲**
2. **打开详情面板**
3. **复制音乐链接**
4. **发送给朋友**
   ```
   "听听这首歌：[粘贴链接]"
   ```

## 💡 使用技巧

### 技巧1：快速复制多个链接

```
1. 点击ℹ️打开详情
2. 依次点击"复制"按钮：
   - 复制音乐链接
   - 复制封面链接
   - 复制歌词链接
3. 粘贴到记事本，一行一个
4. 关闭详情，继续下一首
```

### 技巧2：检查音质

```
1. 在搜索面板选择高码率（740或999）
2. 播放歌曲
3. 打开详情查看"实际码率"
4. 如果实际码率低于请求码率，
   说明该歌曲不支持更高音质
```

### 技巧3：获取高清封面

```
1. 打开详情面板
2. 查看封面预览
3. 复制封面链接
4. 在新标签页打开链接
5. 右键保存图片
```

### 技巧4：调试API

```
1. 打开详情面板
2. 复制歌词API链接
3. 在浏览器新标签页打开
4. 查看API返回的JSON数据
5. 用于学习或开发
```
