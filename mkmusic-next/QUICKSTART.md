# Quick Start Guide

## 快速开始指南

这是 MKOnlineMusicPlayer Next.js 版本的快速启动指南，已集成 GD Studio's Online Music Platform API。

### 📋 前置要求

- Node.js 18.x 或更高版本
- npm 9.0.0 或更高版本
- 互联网连接（访问 GD Studio API）

### 🚀 安装步骤

#### 1. 克隆项目（如果需要）

```bash
git clone <repository-url>
cd mkmusic-next
```

#### 2. 安装依赖

```bash
npm install
```

#### 3. 启动开发服务器

```bash
npm run dev
```

#### 4. 打开浏览器

访问 [http://localhost:3000](http://localhost:3000)

### 🎵 基本使用

#### 搜索音乐

1. 点击页面上的搜索按钮（放大镜图标）
2. 在弹出的搜索面板中：
   - 输入歌手、歌名或专辑名
   - 选择音乐源（网易云/酷我/JOOX）
   - 选择码率（推荐：320kbps）
   - 点击"搜索"按钮

#### 播放音乐

1. 从搜索结果中点击任意歌曲
2. 音乐将自动加载并播放
3. 使用底部的播放器控制：
   - ⏯️ 播放/暂停
   - ⏮️ 上一首
   - ⏭️ 下一首
   - 🔀 随机播放
   - 🔁 列表循环
   - 🔂 单曲循环

#### 查看歌词

- 歌词会在播放时自动显示在右侧面板
- 歌词会随音乐进度自动滚动
- 当前播放的歌词行会高亮显示

#### 分页浏览

1. 搜索结果超过20条时，会显示分页控制
2. 使用方式：
   - 点击"上一页"或"下一页"按钮
   - 使用下拉菜单直接跳转到指定页面
   - 查看当前页码和总页数

### ⚙️ 功能说明

#### 音乐源选择

**网易云 (netease)**
- ✅ 最推荐使用
- ✅ 资源丰富
- ✅ 质量稳定

**酷我 (kuwo)**
- ✅ 备选音乐源
- ✅ 部分独家资源

**JOOX**
- ✅ 国际音乐资源
- ✅ 适合港台音乐

#### 码率说明

| 码率 | 质量 | 文件大小 | 推荐场景 |
|------|------|----------|----------|
| 128kbps | 低 | 小 | 网络受限时 |
| 192kbps | 中 | 中等 | 一般收听 |
| **320kbps** | **高** | **较大** | **日常推荐** |
| 740kbps | 无损 | 大 | 追求音质 |
| 999kbps | 超品 | 很大 | 发烧友 |

💡 **提示**: 默认320kbps在音质和加载速度之间取得良好平衡

### 📱 移动端使用

在移动设备上：
1. 访问相同的URL
2. UI会自动适应小屏幕
3. 所有功能正常工作
4. 支持触摸操作

### ⚠️ 注意事项

#### API限制
- **速率限制**: 5分钟内最多50次请求
- 超过限制会显示错误提示
- 等待5分钟后可继续使用

#### 音乐可用性
- 某些歌曲可能因版权原因无法播放
- 不同音乐源的曲库不同
- 建议尝试不同音乐源

#### 码率支持
- 并非所有歌曲都支持所有码率
- 如果高码率不可用，会自动降级
- 建议优先使用320kbps

### 🐛 常见问题

#### 1. 搜索没有结果
**解决方案**:
- 检查网络连接
- 尝试其他音乐源
- 使用更常见的搜索词

#### 2. 音乐无法播放
**解决方案**:
- 降低码率（如使用320kbps）
- 切换其他音乐源
- 检查浏览器控制台错误信息

#### 3. 请求过于频繁
**解决方案**:
- 等待5分钟
- 避免频繁刷新搜索
- 使用分页而不是反复搜索

#### 4. 歌词不显示
**解决方案**:
- 部分歌曲可能没有歌词
- 尝试播放其他歌曲
- 检查音乐源是否支持该歌曲的歌词

### 📊 性能优化建议

1. **减少请求次数**
   - 使用分页浏览，而不是反复搜索
   - 记住之前的搜索结果

2. **选择合适的码率**
   - 一般情况使用320kbps
   - 网络慢时使用192kbps或128kbps

3. **浏览器选择**
   - 推荐使用最新版Chrome或Firefox
   - 确保浏览器支持HTML5音频

### 🔍 高级功能

#### 播放模式

**列表循环** (默认)
- 播放完当前列表后，从第一首开始

#### 播放列表

- 默认加载云音乐热歌榜
- 可以点击不同的歌单切换
- 搜索结果会创建临时播放列表

### 🎯 最佳实践

1. **搜索策略**
   - 使用歌手+歌名组合搜索更准确
   - 避免使用过于模糊的关键词

2. **播放策略**
   - 首次播放使用320kbps测试

   - 收藏喜欢的歌曲到播放列表

3. **资源使用**
   - 遵守API使用限制
   - 避免短时间内大量请求
   - 合理使用缓存
